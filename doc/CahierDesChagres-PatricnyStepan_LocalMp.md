# Descriptions du projet

le projet consiste a développer une application où l'utilisateur pourra se connecter avec Tiktok, youtube, etc pour pouvoir sélectionner un album et pouvoir télécharger toute les vidéos en Mp3/mp4

# Objectif

MVP = extraire + télécharger vidéos d'un seul service (ex. YouTube) avant d'ajouter TikTok, etc.
